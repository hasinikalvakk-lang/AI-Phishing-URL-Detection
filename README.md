# PhishShield AI - Phishing URL Detection

An AI-powered web application that detects whether a URL is legitimate or potentially malicious using Machine Learning and Flask.

## 🚀 Live Demo

**Live Application:**
[click here](https://z0wh92ft-5000.inc1.devtunnels.ms/)

## 📌 Features

* Detects phishing and legitimate URLs
* Machine Learning-based prediction
* Responsive and modern UI
* Flask backend integration
* Real-time URL analysis
* Cybersecurity-focused project

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Flask
* Python

### Machine Learning

* Scikit-Learn
* Pandas

### Deployment

* Render

## ⚙️ Installation

```bash

cd AI-Phishing-URL-Detection

pip install -r requirements.txt

```
